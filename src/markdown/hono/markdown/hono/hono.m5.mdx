# Hono Deployment Guide

## Introduction

Hono is designed for edge computing, meaning it runs efficiently on platforms like:

### Running on Node.js (Local Server)

For traditional server environments, use Bun or Node.js.

#### Step 1: Install Dependencies

```sh
npm install hono @hono/node-server

```

#### Step 2: Install Dependencies

Create **server.ts**:

```ts
import { Hono } from "hono";
import { serve } from "@hono/node-server";

const app = new Hono();

app.get("/", (c) => c.text("Running on Node.js"));

serve(app);
```

#### Step 3: Install Dependencies

```sh
ts-node server.ts
```

Your Hono app is running on http://localhost:3000

### Deploying on Vercel

Vercel supports Hono as an **Edge Function**.

#### Step 1: Install Vercel CLI

```sh
npm install -g vercel
```

#### Step 2: Create a Hono App

```sh
mkdir hono-vercel && cd hono-vercel
npm init -y
npm install hono

```

#### Step 3: Add Edge Config

Create **vercel.json**:

```json
{
  "functions": {
    "api/index.ts": {
      "runtime": "edge"
    }
  }
}
```

#### Step 4: Deploy

```sh
vercel

```

Your app is live on Vercel Edge Functions.
